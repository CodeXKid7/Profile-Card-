# 🌟 Profile Card with Avatar Upload

A responsive **Profile Card** built with **HTML, CSS, and JavaScript**.  
This project allows users to upload or link a profile avatar, view their info, and see a live-updating clock — all in a clean, modern UI.

---

## ✨ Features

✅ **Responsive Design** — Works on all devices and adjusts layout automatically.  
✅ **Avatar Upload** — Accepts both a URL link or a local image file.  
✅ **Live Clock** — Displays and updates the current time in milliseconds every second.  
✅ **User Info Section** — Includes name, bio, hobbies, and dislikes.  
✅ **Accessible HTML** — Uses semantic structure with `data-testid` attributes for testing.  
✅ **Modern UI** — Rounded corners, smooth hover effects, and a light color palette.  
✅ **Deploy-Ready** — Static files that work instantly on **Netlify**, **GitHub Pages**, or any static host.

## 🚀 How to Run Locally

1. **Download or clone** this repository.  
2. Open the folder and double-click `index.html`.  
3. The page will open in your browser — no server or installation needed.

---

## 🌐 Deploy to the Web

### GitHub Pages
1. Upload the files to a new GitHub repository.  
2. Go to **Settings → Pages**.  
3. Under **Branch**, select `main` and root (`/`).  
4. Wait a few minutes, then visit your live site:  
   `https://yourusername.github.io/repository-name`

### Netlify
1. Go to [https://app.netlify.com/drop](https://app.netlify.com/drop)  
2. Drag and drop the entire project folder.  
3. Netlify will instantly generate a live link.  

---

## 🧠 How It Works

- When a user **enters a URL**, the image preview updates immediately.  
- When a user **uploads a file**, the JS script generates a temporary `blob:` URL and displays it.  
- The **live clock** uses `Date.now()` and updates every second.
